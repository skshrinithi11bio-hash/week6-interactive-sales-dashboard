# Interactive Sales Dashboard

## Project Overview
This project involves building an interactive sales dashboard to visualize sales trends, customer segmentation, and product performance using Python.

The dashboard combines statistical visualizations with interactive charts to provide meaningful business insights.

## Objectives
- Analyze sales trends over time
- Compare product category performance
- Visualize statistical distributions
- Create interactive dashboards using Plotly

##  Tools & Libraries
- Python
- Google Colab
- Pandas
- Seaborn
- Matplotlib
- Plotly

## Setup Instructions
1. Open Google Colab
2. Mount Google Drive
3. Place `sales_data.csv` in Drive
4. Run `dashboard.ipynb`


## Visualizations Included
- Box plot
- Violin plot
- Line chart
- Correlation heatmap
- Interactive bar chart
- Interactive pie chart


## Key Insights
- Product categories vary significantly in price distribution
- Sales trends show time-based patterns
- Interactive charts allow better data exploration

---

## Conclusion
The project successfully demonstrates an interactive sales dashboard using both static and interactive visualizations to support data-driven decision-making.
